# Learning OpenGL Compute - Ray Tracing in a Weekend

My personal learning experience based on https://raytracing.github.io/

Trying to get a simple ray tracer done in OpenGL 4 compute shaders.

## Build Environment

* CMake
* Visual Studio 2017

## Dependencies

 * OpenGL 4.5
 * GLM
 * GLFW
 * GLEW
 * G3LOG
